## Run Locally

Clone the project

```bash
  git clone https://github.com/tarifalhasan/Crew-Labs
```

Go to the project directory

```bash
  cd Crew-Labs
```

Install dependencies

```bash
  npm install
  or
  yarn

```

Start the server

```bash
  npm run dev
  or yarn run dev
```
